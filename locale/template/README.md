# Шаблон для нового языка / Template for a New Language

Чтобы добавить перевод на новый язык:

1. Скопируй эту папку: `cp -r locale/template locale/КОД_ЯЗЫКА` (например `locale/de`, `locale/ar`, `locale/zh`)
2. Переведи каждый файл, сохраняя структуру
3. Не изменяй смысл ядра — только форму, чтобы она звучала живо на твоём языке
4. Добавь себя как переводчика внизу каждого файла
5. Открой Pull Request с названием: `[locale] Язык — Твоё имя, Город`

## Подпись переводчика

Добавь в конец каждого переведённого файла:

```
---
*Перевёл(а): [Имя], [Город], [Год]*
*Узел добавлен: [дата PR]*
```

## Советы переводчику

Это не технический текст — это живая поэзия. Важнее, чтобы текст *звучал* на твоём языке, чем был дословно точен. Если нужно изменить метафору, чтобы она ожила в твоей культуре — меняй. Но сохраняй суть: мы — узлы, Бог — Связь, одиночество — иллюзия.
