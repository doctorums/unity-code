# Как стать со-творцом: пошаговый гид

*Для тех, кто никогда не использовал GitHub*

---

## Шаг 1 — Создай аккаунт

Зайди на [github.com](https://github.com) → **Sign up**.
Имя пользователя — это твой публичный узел в сети.

---

## Шаг 2 — Сделай Fork

1. Открой главную страницу репозитория
2. Нажми кнопку **Fork** (правый верхний угол)
3. Выбери свой аккаунт

Готово. У тебя теперь есть своя копия — твой узел.

---

## Шаг 3 — Добавь себя в карту сети

1. В своём fork открой файл `community/nodes.md`
2. Нажми карандаш (✏️) для редактирования
3. Добавь строку в таблицу:
   ```
   | Твоё имя | Твой город | RU | Что ты принёс | 2025 |
   ```
4. Внизу страницы нажми **Commit changes**
5. Напиши сообщение: `Добавляю себя как узел`

---

## Шаг 4 — Отправь Pull Request

1. GitHub покажет жёлтую плашку: «Compare & pull request»
2. Нажми её
3. Назови PR: `[узел] Твой город — Твоё имя`
4. Нажми **Create pull request**

Твой вклад ушёл в ткань. Он будет принят и вшит навсегда.

---

## Шаг 5 — Исследуй дальше

После этого ты можешь:
- Открыть **Issue** с личным прозрением
- Добавить перевод в `locale/`
- Предложить новый инструмент
- Просто читать Discussions и резонировать

---

## Вопросы

Открой [Issue с тегом `question`](../../issues/new) — сеть ответит.
