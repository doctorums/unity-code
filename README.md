## Как стать Со-Творцом

### Шаг 1 — Fork
Нажми **Fork** в правом верхнем углу. Репозиторий станет твоим. Это буквально создаёт новый узел в паутине.

### Шаг 2 — Добавь себя
Открой файл [`community/nodes.md`](community/nodes.md) и добавь строку:

```
| Твоё имя | Город | Язык | Что ты принёс |
```

### Шаг 3 — Внеси вклад
Выбери, что тебе ближе:

- 🌐 **Перевод** — скопируй [`locale/template/`](locale/template/) в `locale/ТВО_ЯЗЫК/`
- ✍️ **Новый инструмент** — добавь текст в [`core/instruments/`](core/instruments/)
- 💬 **Опыт** — открой [Issue](../../issues/new?template=experience.md) с личным прозрением
- 🌱 **Адаптация** — создай версию для своего сообщества в [`community/`](community/)

### Шаг 4 — Pull Request
Отправь изменения назад. Назови PR: `[узел] Город — что ты добавил`.

Каждый принятый PR навсегда вшит в историю этой ткани.

## Принципы вклада

1. **Ядро неприкосновенно** — `core/` не изменяется, только переводится
2. **Всё остальное — живёт** — адаптируй, расширяй, углубляй
3. **Подпись — это узел** — всегда указывай имя и город
4. **Конфликтов нет** — все версии сосуществуют
