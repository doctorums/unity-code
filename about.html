Что это?

UnityCode 7.6 — это протокол превращения экзистенциального хаоса в интеллектуальные резонансы. Вы вводите любые обрывки мыслей, страхи, интуиции («шум»). Алгоритм (LLM) не оценивает и не имитирует чувства, а ищет паттерны и связи с наукой, философией, математикой, этикой. Результат — структурированная интерпретация и вопрос для рефлексии. Если ответ откликается — вы «вплетаете» узел в общую сеть, и другие могут добавлять свои слои (дочерние узлы).

«Твой сигнал резонирует с такой-то концепцией. Если это откликается — вот возможный путь интеграции.»

Децентрализация: сеть порталов

UnityCode — не один сайт, а открытый протокол. Любой желающий может скопировать ядро (5 страниц + API) и запустить свой узел-портал на любом хостинге (GitHub Pages, Vercel, собственный сервер). Порталы могут обмениваться сигналами через единый API, создавая распределённую паутину. Никто не владеет всей сетью — сеть принадлежит протоколу.

┌─────────────┐ ┌─────────────┐ ┌─────────────┐ │ Мастер-узел │◄────►│ Порталы │◄────►│ Ваш личный │ │ (master) │ sync │ друзей │ sync │ узел │ └─────────────┘ └─────────────┘ └─────────────┘ ▲ ▲ ▲ └───────────┬──────────┘ │ │ │ обмен узлами (API) создание дочерних на чужих узлах

Как создать свой портал?

  1. Форкнуть репозиторий на GitHub.
  2. Настроить Supabase (БД) по инструкции.
  3. Заменить API-ключи в config.js.
  4. Задеплоить на Vercel / Netlify / GitHub Pages.
  5. Готово — ваш узел в сети!

Технологии (малозатратный стек)

  • AI-ядро: Google Gemini API / Cloudflare Workers AI (бесплатные лимиты)
  • База данных: Supabase (бесплатно до 500 МБ) или PocketBase
  • Хостинг статики: GitHub Pages / Vercel (бесплатно)
  • Карты: Leaflet + OpenStreetMap (бесплатно)
  • Бэкенд-прослойка: Edge Functions (Netlify / Vercel / Cloudflare Workers)

Метрики успеха

  • Количество созданных узлов.
  • Количество дочерних связей на один родительский узел.
  • Глубина ветвления (поколений узлов).
  • Возвраты пользователей.
  • Качественные отзывы: снижение экзистенциального стресса, чувство связи.

Критерии этики

Анонимность. Жёсткий фильтр экстремизма. Никаких персональных данных. Юридический дисклеймер: некоммерческий научно-художественный эксперимент, контент не претендует на объективную истинность.
