## О проекте

UNITYCODE — протокол, который превращает внутренний хаос в резонирующие интеллектуальные модели.  
Если модели снижают экзистенциальный стресс и укрепляют чувство связи — **протокол работает**.

Версия: **7.6 (V6)** | Жанр: Генеративный синтез | Лицензия: MIT

## ИИ-интеграция (Петля)

`petlya.html` поддерживает два режима:

**Демо-режим** (без ключа):  
Встроенные ответы рефлексивного узла. Полностью рабочий, без API.

**Режим с API:**  
Вставь свой Anthropic API ключ (`sk-ant-...`) в поле на странице.  
Ключ хранится только в браузере, не отправляется на сервер.

## Принципы

- **Шум — это материал.** Хаос — необработанный сигнал, ожидающий интеграции.
- **Нет истин, только резонансы.** Система говорит «твой сигнал резонирует с...»
- **Каждый узел порождает узлы.** Смыслы наслаиваются, паутина растёт.
- **Анонимность — защита сигнала.** Никаких персональных данных. Только сигнал.
